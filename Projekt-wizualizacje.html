Wstęp

Celem niniejszego opracowania jest analiza oraz wizualizacja danych dotyczących ruchu rowerowego w Gdańsku w oparciu o system automatycznych punktów pomiarowych. Projekt skupia się na zbadaniu dynamiki przejazdów, identyfikacji wzorców czasowych oraz określeniu wpływu warunków atmosferycznych na aktywność rowerzystów.

W zbiorach danych znajdują się informacje o dziennych liczbach przejazdów rowerowych w wybranych punktach oraz informacje o warunkach pogodowych w danym dniu oraz współrzędne geograficzne punktów pomiarowych.

Zmienna Opis

stacja_wybrana <- "ul. Chłopska"
dane_stacja <- przejazdy[przejazdy$Stacja == stacja_wybrana, ]
par(mar = c(5, 4, 4, 2) + 0.1)

hist(dane_stacja$Licznik, 
     breaks = 20,
     main = paste("Rozkład liczby przejazdów:", stacja_wybrana),
     xlab = "Liczba przejazdów dziennie",
     ylab = "Częstość (liczba dni)",
     col = kolorki[6],
     border = "white")

ggplot(dane_stacja, aes(x = Licznik)) +
  geom_histogram(bins = 20, 
                 fill = "#C51B7D", 
                 alpha = 0.7) +
  labs(title = paste("Rozkład liczby przejazdów -", stacja_wybrana),
       x = "Liczba przejazdów dziennie",
       y = "Gęstość") +
  theme_light() +
  theme(plot.title = element_text(hjust = 0.5),
        plot.subtitle = element_text(hjust = 0.5))

Porównanie punktów pod względem natężenia /rozkładu przejazdów

stacje_order <- aggregate(Licznik ~ Stacja, data = przejazdy, median)
stacje_order <- stacje_order$Stacja[order(stacje_order$Licznik)]

przejazdy$Stacja_f <- factor(przejazdy$Stacja, levels = stacje_order)

Stacje posortowane według mediany dziennej liczby przejazdów.
